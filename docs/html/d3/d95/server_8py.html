def server.update_minus_steer_pwm (   scale)

ステアリング制御用(左方向)の関数

http://localhost.local:<PORT>/steer/-<scale>/ の<scale>に値を設定して、httpリクエストを送ることで呼び出しされる関数。呼び出しに成功するとステアリング用に設定したGPIOピンから、PWM出力が行われる。

警告
httpリクエストする際は, スケールの部分に"-"をつけるのを忘れないように
引数
[in]floatscale ステアリングのスケール値を0.0~1.0(正面方向~左方向)で指定。
